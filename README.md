# 🎮 Game Glitch Investigator: The Impossible Guesser

## 📌 Project Overview

This is a simple number guessing game built with Streamlit. The starter version
was generated by AI and shipped full of bugs, so the goal of this project was to
investigate and fix them. The game is now fully playable: it picks a secret
number, takes your guesses, and gives correct feedback until you win or run out
of attempts. It correctly handles winning guesses, too-high and too-low guesses,
hints about the secret number, a "New Game" reset, and warnings for out-of-range
guesses.

## 🛠️ Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python -m streamlit run app.py`

## 📸 Demo Walkthrough

1. Run `python -m streamlit run app.py` and open the app in your browser.
2. Pick a difficulty in the sidebar to set the range and number of attempts.
3. Type a number into the guess box and click **Submit Guess 🚀**.
4. Read the feedback: a too-low guess says "Go Higher!", a too-high guess says
   "Go Lower!", and a correct guess shows a win message with your score.
5. Click **Show Hint 💡** to reveal a hint about the secret number (whether it is
   even or odd and which half of the range it falls in).
6. Try an out-of-range number (like `150`): the game shows a clear warning and
   does not count it as an attempt.
7. Click **New Game 🔁** to fully reset the game — a new secret number, and the
   score, attempts, and history all start fresh.

## 🐛 Bugs Fixed

- The Higher/Lower hints were backwards — now too-low says "Go Higher!" and
  too-high says "Go Lower!".
- The secret number was being compared as a string; it now compares as a number.
- The "New Game" button now fully resets the secret, score, attempts, history,
  and game status.
- The "Show Hint" button now actually displays a hint that persists across reruns.
- Out-of-range guesses now show a warning instead of being processed as a normal guess.
